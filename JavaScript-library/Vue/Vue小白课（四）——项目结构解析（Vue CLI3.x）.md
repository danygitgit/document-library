# 正文

&emsp;vue-cli3.0默认项目目录与2.0的相比，更加精简:
1. 移除的配置文件根目录下的，build和config等目录，
2. 移除了static文件夹，新增了public文件夹，并且index.html移动到public中。
3. 在src文件夹中新增了views文件夹，用于分类 试图组件 和 公共组件 。
4. 大部分配置 都集成到 vue.config.js这里,在项目根目录下
